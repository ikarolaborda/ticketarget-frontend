# TicketArget — Frontend

Vue 3 + Pinia + Vite SPA: browse with infinite scroll, fuzzy search with
autocomplete, seat-map selection, waiting-room UX with auto-retry, and Stripe
checkout with a live hold countdown.

Part of the [TicketArget platform](https://github.com/ikarolaborda/ticketarget) —
built into an nginx image by the aggregator (`make frontend-build`).
